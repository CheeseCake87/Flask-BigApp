Blueprint importing

If you don't have name defined in the Blueprint config file, it will set the endpoint to the name of the folder the Blueprint is in.

Being able to set this name can avoid any endpoint duplicates if you have are importing blueprints from different folders.

Structures (themes) are blueprints to.

When importing structures, these are the same as blueprint in the way they work, although they have more of a focus on controlling the page layout and styling. Structures usually don't have any routes assigned
